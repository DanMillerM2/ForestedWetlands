Build Random Forest

Title  Build Random Forest

Description

          

            

              

                

                  Build a random forest model to predict probability of wetland occurrence. Probability is determined as a function of the specified set of input rasters, which serve as explanatory variables for the model, and a set of point locations with verified wetland class, which serve as dependent variables for training the model. This script is written in R and uses the randomForest package. 
                

                

                  Output is written to the "View Details" window and includes two useful measures of model performance: the "Out-of-bag" (OOB) error rate and a confusion matrix, which shows how many of the training data points were correctly classified, how many non-wetland points were incorrectly classified as wetland, and how many wetland points were incorrectly classified as non-wetland. These measures of error indicate the degree to which information provided by the input rasters can delineate wetland and non-wetland areas based on the input data. These measures provide a means of comparing model performance for different sets of input rasters. Another useful output is a list showing the relative importance of each input raster for determining wetland class (this information is also provided in a plot).
                

                

                  Note that the input rasters are renamed as Raster1, Raster2, etc. in the order that they were specified.

Syntax

Parameter Explanation Data Type
Work_Space Dialog Reference

This is the ArcGIS workspace. Click on the dialog icon to the right to select the folder to serve as the workspace.

There is no python reference for this parameter.

Folder
Input_Rasters Dialog Reference

A set of rasters that provide information to infer probability of wetland occurrence. These rasters must all have the same extent, cell size, cell-corner locations, and coordinate system. A set of rasters derived from a single DEM using the Surface Metrics script tool in the DEMutilities toolbox will meet these criteria; rasters from other sources need to be resampled within ArcGIS if they do not meet these criteria. Click on the folder icon to the right to pull up a dialog box from which to select input rasters.

There is no python reference for this parameter.

Multiple Value
Training_Data Dialog Reference

A point feature class containing locations classified as wetland or non-wetland. Click on the folder icon at the left to select a feature class.

There is no python reference for this parameter.

Feature Class
Input_Field Dialog Reference

The field of the training-data feature class with the wetland class labels.

There is no python reference for this parameter.

Field
Wetland_Class Dialog Reference

The entry in the Input Field that indicates the point is a wetland.

There is no python reference for this parameter.

String
Not-a-wetland_Class Dialog Reference

The entry in the Input Field that indicates the point is not a wetland.

There is no python reference for this parameter.

String
Model_name Dialog Reference

The random forest model is stored in the working directory as model_name.RFmodel, where "model_name" is this field's value. The list of input-raster-file names is also stored as model_name.RasterList

There is no python reference for this parameter.

String
Output_probability_raster (Optional) Dialog Reference

The name of the optional output probability raster file. The output raster is stored as a GeoTiff file called raster_name.tif, where "raster_name" is this field's value. Depending on the class names for Wetland and Upland, the probability raster shows either the modeled probability that a wetland will exist within each DEM cell (if the wetland class name is alphabetically before the upland class name), or the probability that a wetland does not exist in each DEM cell. If the raster shows the probability of not being a wetland, simply subtract it from one in the raster calculator to get the probability of being a wetland.

There is no python reference for this parameter.
